## ⚡ What I actually spend my time on

I'm not the "I know 20 languages" type — I go deep on **making AI agents reliable enough to trust in production**. That means retry logic that doesn't silently swallow errors, RBAC that actually isolates data, and prompts with guardrails so the LLM doesn't go rogue.

- 🏫 Right now: a **live bilingual school platform** where n8n + Gemini handle grading, result analysis, and parent communication automatically
- 🧠 Also built: a **RAG chatbot** that's only allowed to answer from its own knowledge base — no hallucinated facts, ever
- 📊 Before automation, I was elbow-deep in **classic ML** — models that predict churn, income, and health risk from messy real-world data
- 🏆 Placed **top 8%** (11th/150 teams) in the Qafza × IEEE AI Automation Competition
